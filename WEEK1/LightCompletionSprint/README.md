# 2. One Learning from Each Module

## 1. Salesforce Values: Quick Look

**Learning:** I learned about Salesforce core values like trust, customer success, innovation, equality, and sustainability.

## 2. Trailhead Playground Management

**Learning:** I learned how to create and manage Trailhead Playgrounds for practice and hands-on challenges.

## 3. Salesforce Developer: Quick Look

**Learning:** I learned the role of a Salesforce Developer and how developers customize Salesforce applications.

## 4. Salesforce CRM

**Learning:** I learned how Salesforce CRM helps organizations manage customer relationships and business processes.

## 5. Agentforce 360 Platform Basics

**Learning:** I learned the fundamentals of the Agentforce 360 Platform and its business capabilities.

## 6. Agentforce 360 Platform Development Basics

**Learning:** I learned how development tools are used to customize and extend the Agentforce platform.

## 7. Agentforce 360 Platform Events Basics

**Learning:** I learned how platform events enable event-driven communication in Salesforce.

## 8. Apex & .NET Basics

**Learning:** I learned the basics of integrating Apex with .NET technologies.

## 9. Formulas and Validations

**Learning:** I learned how formula fields and validation rules improve data accuracy and automation.

## 10. Data Modeling

**Learning:** I learned how objects, fields, and relationships are used to design a structured database.

## 11. Flow Builder Basics

**Learning:** I learned how Flow Builder automates business processes without coding.

## 12. Data and Actions in Flows

**Learning:** I learned how to use records, decisions, and actions inside Salesforce Flows.

## 13. Database & .NET Basics

**Learning:** I learned the fundamentals of databases and their connection with .NET applications.

## 14. Search Solution Basics

**Learning:** I learned how search solutions improve information retrieval and user experience.

## 15. Command-Line Interface

**Learning:** I learned how command-line tools help developers manage Salesforce projects efficiently.

## 16. Apex Basics & Database

**Learning:** I learned how Apex interacts with Salesforce databases using SOQL and DML operations.

## 17. Apex Triggers

**Learning:** I learned how Apex Triggers automate actions before or after database events.
