## What is LWC?

**Lightning Web Components (LWC)** is Salesforce’s modern framework for building fast and reusable user interfaces using **HTML, CSS, and JavaScript**.

## Why Salesforce Uses LWC

* Faster performance
* Reusable components
* Uses standard web technologies
* Easy integration with Salesforce data and Apex
* Better user experience

## Your UI Screens (College Management System)

1. **Student Registration** – Add new students.
2. **Course Management** – Manage courses.
3. **Attendance Tracking** – Mark attendance.
4. **Student Dashboard** – View student details and progress.
5. **Reports** – Show enrollment and attendance statistics.

## Component Breakdown

* `studentRegistration` – Register students.
* `courseManagement` – Manage courses.
* `attendanceTracker` – Record attendance.
* `studentDashboard` – Display student information.
* `collegeReports` – Generate reports.

## Frontend vs Backend Logic

**Frontend (LWC):**

* Displays forms and pages
* Handles user interactions
* Validates input

**Backend (Apex):**

* Processes business logic
* Queries and updates Salesforce records
* Performs validations and calculations

## Reflection

LWC helps create responsive and user-friendly Salesforce applications. It improves performance, supports reusable components, and integrates easily with Apex and Salesforce data.
